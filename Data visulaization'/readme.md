# 📊 Data Visualization Dashboard

An interactive and responsive Data Visualization Dashboard built using HTML, CSS, and JavaScript. The project loads data from a JSON file and displays it in a clean, user-friendly interface for better understanding and analysis.

---

## 🚀 Features

- 📈 Interactive data visualization
- 📂 JSON data integration
- 📱 Responsive design
- ⚡ Fast loading
- 🎨 Modern UI
- 🔄 Dynamic content rendering
- 🌙 Clean and attractive interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- JSON

---

## 📂 Project Structure

```
Data Visualization/
│
├── index.html
├── style.css
├── app.js
├── data.json
└── README.md
```

---

## 💻 How to Run

1. Download or Clone the repository
2. Open the project folder
3. Open **index.html** in your browser

No installation required.

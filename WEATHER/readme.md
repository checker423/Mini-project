# 🌦️ Weather App

A modern Weather Application built using HTML, CSS, and JavaScript that provides real-time weather information for any city using a weather API.

---

## 🚀 Features

- 🌍 Search weather by city
- 🌡️ Current temperature
- 💧 Humidity
- 🌬️ Wind Speed
- ☁️ Dynamic weather icons
- 📱 Responsive Design
- ⚡ Real-time weather updates

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API

---

Weather/

---

## 💻 How to Run

1. Clone or Download this repository
2. Add your Weather API Key inside **app.js**
3. Open **index.html** in your browser

# 📋 Project Management Dashboard

A responsive Project Management Dashboard developed using HTML, CSS, and JavaScript. It helps users organize projects, manage tasks, monitor progress, and improve productivity through a clean and intuitive interface.

---

## 🚀 Features

- 📌 Task Management
- 📊 Project Overview
- 📅 Progress Tracking
- 📱 Responsive Design
- 🎨 Modern Dashboard UI
- ⚡ Fast Performance
- 📝 User-friendly Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
